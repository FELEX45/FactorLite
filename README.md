# FactorLite

2D survivor-like таймкиллер под Android (libGDX + Kotlin): игрок двигается через floating joystick, персонаж авто-атакует ближайшего врага, прогресс через Level Up (оружия/пассивки) и сундуки.

## Запуск Desktop (для быстрой отладки)

```powershell
cd C:\Users\Professional\Desktop\51A2~1
.\gradlew.bat :lwjgl3:run
```
