# Docs

## Content editor

Открой `docs/content_editor.html` двойным кликом (локально, без сервера).

Он позволяет:
- добавлять **оружия / пассивки / предметы (сундуки) / бонусы шрайна**
- получать итоговый **YAML/JSON** (справа), копировать или скачивать

Данные сохраняются в `localStorage` браузера (на твоём ПК).

## Seed-черновик по оружию

Я собрал стартовый черновик по описанию оружия из `docs/мекасронк.txt`:
- `docs/factorlite_content_seed.yaml`

Это пока **документ/шаблон** (игра его ещё не читает напрямую), но он помогает быстро согласовать:
- базовые статы
- список апгрейдов на оружие
- лимиты (cap) и относительные шансы (weight)
