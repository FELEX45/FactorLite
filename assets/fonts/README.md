# Fonts

Для корректной **кириллицы** на Android/PC положи сюда шрифт:

- Рекомендуемое имя: `Roboto-Regular.ttf`
- Путь в коде: `Gdx.files.internal("fonts/Roboto-Regular.ttf")`

Почему файл не лежит тут автоматически:
- это бинарный файл; проще добавить его вручную и закоммитить один раз.

Проверка:
- если шрифт есть, UI будет с кириллицей;
- если шрифта нет, игра упадать не будет, но может быть “квадратики/латиница”.
