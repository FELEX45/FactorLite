# Textures

Плейсхолдер для графики.

План:
- иконки оружий/пассивок/предметов
- простые спрайты врагов/пуль
- фон арены

## Имена файлов (PNG)

Клади PNG сюда: `assets/textures/<name>.png`

Сейчас игра ищет (если файла нет — рисует плейсхолдер):
- `bg_forest.(png/jpg/jpeg)`, `bg_desert.(png/jpg/jpeg)`, `bg_winter.(png/jpg/jpeg)` (фон арены по сложности)
- `player.png` (общий фоллбек)
- `player_frozka.png`, `player_farzer.png`, `player_joe.png`, `player_trapper.png`, `player_nikumi.png`, `player_sapka.png`, `player_zhidj.png`
- `enemy_normal.png`, `enemy_fast.png`, `enemy_tank.png`, `enemy_ranged.png`
- `enemy_elite.png`, `enemy_boss.png`
- `projectile_player.png`, `projectile_enemy.png`
- `orb_xp.png`, `orb_gold.png`
- `chest.png`, `chest_elite.png`
- `shrine.png`
- `cloud_toxic.png`, `trap.png`
